# hehsfesfe

## HeroUI Native 官方示例 App（Android）

已基于 [heroui-inc/heroui-native-example](https://github.com/heroui-inc/heroui-native-example) 构建 Android Debug 包，可直接下载安装体验组件效果。

| 项目 | 说明 |
|------|------|
| 应用名 | HeroUI Native Example |
| 包名 | `com.anonymous.herouinativeexample` |
| 版本 | 1.0.0 |
| 构建类型 | Debug（用于预览 UI 组件） |
| 架构 | arm64-v8a |

### 下载

**直接下载 APK：**

[app-packages/heroui-native-example-debug.apk](./app-packages/heroui-native-example-debug.apk)

GitHub 原始链接：

`https://github.com/cutecat2331234/hehsfesfe/raw/main/app-packages/heroui-native-example-debug.apk`

### 安装说明

1. 将 APK 下载到 Android 手机
2. 允许「安装未知来源应用」
3. 点击安装即可

> 这是 Debug 包，仅用于预览 HeroUI Native 组件，不是 Google Play 正式版。
