# hehsfesfe

## HeroUI Native 官方示例 App（Android）

已基于 [heroui-inc/heroui-native-example](https://github.com/heroui-inc/heroui-native-example) 构建 **Release 独立安装包**（JS 已内嵌，无需联网开发服务器）。

| 项目 | 说明 |
|------|------|
| 应用名 | HeroUI Native Example |
| 包名 | `com.anonymous.herouinativeexample` |
| 版本 | 1.0.0 |
| 构建类型 | **Release**（可离线独立运行） |
| 架构 | arm64-v8a + armeabi-v7a |

### 下载

**直接下载 APK：**

[app-packages/heroui-native-example.apk](./app-packages/heroui-native-example.apk)

GitHub 原始链接：

`https://github.com/cutecat2331234/hehsfesfe/raw/main/app-packages/heroui-native-example.apk`

### 安装说明

1. 将 APK 下载到 Android 手机
2. 允许「安装未知来源应用」
3. 点击安装即可

> 之前上传的 Debug 包未内嵌 JS，会卡在启动页；请使用本 Release 包。

### App 里能看到什么？

打开后可浏览 **HeroUI Native 全部官方组件示例**（约 38 个），包括：

Accordion、Alert、Avatar、BottomSheet、Button、Card、Checkbox、Chip、Dialog、Input、InputOTP、Menu、Popover、Select、Slider、Switch、Tabs、Toast、Typography 等。

另有 **Showcases**（付费墙、Raycast 风格、Linear 任务等场景演示）和 **Themes** 主题页。

官方源码：https://github.com/heroui-inc/heroui-native-example
